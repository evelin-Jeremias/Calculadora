 # Calculadora 2024

Uma calculadora de console simples mas poderosa que permite realizar as quatro operações matematicas 

## Funcionalidades

- *Operações Básicas*: somas, multiplicações e divisões com facilidade.
- *Suporte a Decímais*: Trabalhe com números que têm até duas casas decimais.
- *Tratamento de divisões por Zero*: A calculadora é capaz de validar erros de divisão por zero.

---

## Como usar:

1. Clone o repositório ou baixe o código donte da calculadora.
2. Abra ou terminal ou prompt de comando e navegue até a pasta raiz da calculadora.
3. Utilize o comando abaixo para restaurar as dependecias do projeto.
```
dotnet restore
```

4. Em seguida, compile a solução do comando:
```
dotnet buid -- configuration Release
```
5. Para executar o ppojeto compilado em temp real
```
dotnet run --project Calculadora, 
```
6. ´Para executar o arquivo compilado, navegue até a pasta ./Calculadora.ConsoleApp/bin/Release/net8.0/

.NET SDK (recomendado .NET 5.0 ou superior) para compilação e execução do projeto.
